    About This Project

    A choropleth map helps to visualize how a certain statistic or measurement varies across a geographic region. I implemented this project as part of FreeCodeCamp's Data Visualization certificate course. This particular project features data on the percentage of Americans with a bachelor's degree or higher by county in the United States.
